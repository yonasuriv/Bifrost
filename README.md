![Bifrost Link](https://github.com/yonasuriv/bifrost/assets/59540565/08a97b1b-e112-4aef-9d1d-b658b1390785)

## Installation
1. ```git clone https://github.com/yonasuriv/bifrost.git```
2. ```sh bifrost.sh```
3. Press ```X``` to install all the dependencies
4. Enjoy
