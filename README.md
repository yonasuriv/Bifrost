### Installation
1. ```git clone https://github.com/yonasuriv/bifrost.git```
2. ```sh bifrost.sh```
3. Press ```X``` to install all the dependencies
4. Enjoy

Want to contribute? Suggest any change/update/upgrade/feature or just fork the project.
